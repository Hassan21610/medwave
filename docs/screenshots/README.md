# Screenshot Guide

Add showcase images here so they can be referenced by the main `README.md`.

Recommended files:

- `dashboard.png` - main app workspace with camera preview and PDF viewer
- `auth-window.png` - login and face enrollment screen
- `gesture-guide.png` - gesture guide and voice status panel

Tips:

- Use 16:9 or 4:3 screenshots for clean rendering on GitHub.
- Blur any sensitive data before uploading.
- Keep each file under 2 MB for faster page load.
